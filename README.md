### 프로그램 개요
- 프로그램 : `사용자의 요구`에 따라 `데이터`를 `처리`(표현, 이동, 저장, 로드 등)를 할 수 있는 `소프트웨어`
- 프로그램을 만드는 작업 : 프로그래밍(코딩, SW개발, 소프트 엔지니어 등)
